About Us As The Creators

  • CCP Alarms is a 2nd year project By Colin Smith, Conor O'Dea and Padraig Nolan Hence CCP.

    • CCP Alarms - What the project is about?

  • This project is a hub showing all alarm changing status aimed for a admin point of view from which, you as an admin can filter households via Postcode. Each house has its unique statuses based on each sensor status in which if a sensor is on the row is green the sensor status is on if it's not on the row is red.
